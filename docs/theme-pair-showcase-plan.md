# Theme Pair Showcase Plan

- Create a dedicated theme registry with 5 named visual directions.
- Give each visual direction a complementary `day` and `night` token set based on shadcn-style CSS variables.
- Persist the selected theme pair and active mode in browser local storage.
- Add a compact selector panel in the workspace shell so each theme pair can be previewed from the UI.
- Keep the current workspace layout intact while letting the active theme drive shared tokens and atmosphere.
- Add tests around theme normalization and selector rendering.
